# 此刻 · Still

> 一个让自己慢下来、回到当下的私人空间。
> 记录情绪、练习呼吸、照看自己——所有数据只留在你的设备上。

「此刻」是一个纯前端的冥想与情绪记录应用。没有账号，没有云端，不收集任何数据，打开就用，离线也能用。它被做成可以「添加到主屏幕」的网页应用（PWA），在 iPhone 上用起来和原生 App 几乎一样。

## 它能做什么

- **此刻** —— 记录当下的情绪。先选能量与愉悦的方向，再选具体感受，可以标注强度与情境，写下只言片语。每次记录后，会收到一张温柔的鼓励卡。
- **课程** —— 系统学习、情绪急救、概念涨知识三类内容。
- **冥想** —— 9 个分类（呼吸练习、身体扫描、正念觉察、助眠放松、感恩冥想、情绪疗愈、专注力、晨间唤醒、晚间回顾），共三十余个练习。呼吸练习带动态的节奏引导。
- **成长** —— 一棵随记录次数慢慢长大的小树，配合情绪打卡日历、本月心情、情境与情绪的关系、周/月报告，以及数据导出。
- **时刻卡** —— 上传自己的图片、起个名字，做成属于你的卡片。
- **主题** —— 10 种点缀色可选，也可以开启「心情变色」，让主色跟随你最近一次情绪自动变化。

## 技术说明

- 单个 `index.html`，无框架、无构建、无外部依赖。
- 数据存在本地：文字记录用 `localStorage`，图片与音视频用 `IndexedDB`。
- 内置 JSON 完整备份与恢复。
- 不联网，不接入任何统计或追踪。

## 部署（GitHub Pages）

1. 新建一个 **Public** 仓库，上传 `index.html`（可选 `assets/`）。
2. **Settings → Pages**，Source 选 *Deploy from a branch*，分支选 `main`、目录 `/(root)`，保存。
3. 等 1–2 分钟，得到网址 `https://<用户名>.github.io/<仓库名>/`。
4. 用 **iPhone Safari** 打开 → 分享 → **添加到主屏幕**。

> 装到主屏幕后，数据会以 PWA 形式持久保存，不受 iOS 定期清理的影响。

## 放自己的封面与音频（可选）

在 `assets/` 里按课程 id 命名即可自动匹配：

- 封面图：`assets/<id>.jpg`
- 音频：`assets/<id>.mp3`

不放也没关系，应用内置了默认封面与白噪音。完整的课程 id 一览见 `README.txt`。

> 语音引导音频需自行准备（推荐用 TTS：微软 Azure `zh-CN-XiaoxiaoNeural` 语速 -20%，或火山引擎、CosyVoice 等），做好后命名上传即可。本应用不自带人声朗读。

## 数据与隐私

所有内容只存在你自己的浏览器里。没有服务器，没有账号，没有分析统计。在卸载或清除浏览器数据前，记得用「导出完整备份」留存一份。
